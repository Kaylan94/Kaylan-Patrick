My Work

During my time at HyperionDev, I worked on many projects and got to learned and understand the fundamentals of Software Engineering. Below are some of the projects that I have completed.

Poised Project Management (Java + MySQL)

In this project I used Java and MySQL to build a functional Project Management tool. The tool incorporates the following features:

  • Initializing/capturing project Information
  • Capturing of stakeholder information
  • Database for the storage of all information
  • Updating information on current projects
  • Updating information on current stakeholders
  • Finalising projects and invoice generation

Task Manager (Python CRUD Application)

In this project I used Python to build a functional Task Manager. The tool incorporates the following features:

  • Login Authentication (User and Password matching)
  • User creation - All user information stored in an external text file(this action made available only to the Admin user only)
  • Task creation - This involve the capturing of task info, assigning a task to an existing user, and
  • the storage of this info to external text file as well.
  • Updating information involving the reading and updating of information in the text file
  • Displaying statistics such as tasks overdue, tasks per user, outstanding tasks per user and tasks completed per user
  • Functionality for regular users limited. Complete functionality available for Admin. Thus, the menu displayed and options available were user dependent.
  • Menu options included: 'View all tasks', 'View my tasks', 'Register User', 'Add task', 'Statistics', 'Exit'
